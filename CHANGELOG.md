# Changelog

## 0.2.3 - 2026-06-21

- 新增 YouTube 正在直播时的聊天弹幕，新增聊天会即时显示。
- 新增直播回放聊天弹幕，按聊天发送时间映射到视频时间线。
- 支持直播聊天中的表情图片弹幕。
- 优化直播中、直播回放和普通视频的识别逻辑，避免页面内跳转后误加载旧视频弹幕。
- 修复直播回放刚打开时聊天弹幕集中挤满，以及拖动进度后聊天弹幕消失的问题。
- README 增加直播聊天弹幕支持说明。

## 0.2.2 - 2026-06-14

- 修复控制面板弹幕开关和视频页弹幕按钮状态不同步的问题。
- 控制面板保存设置时只写入变更字段，避免旧状态覆盖页面按钮状态。
- 修复扩展重新加载后旧页面脚本访问 storage 时的 `Extension context invalidated` 报错。
- 修正 GitHub Release 和 Packages 元数据里的 YouTuBi 名称大小写。

## 0.2.1 - 2026-06-14

- 优化长弹幕移动速度，减少长弹幕被强制加速的情况。
- 回复面板里的用户名统一使用蓝色展示。
- 修正 README、扩展页和弹窗里的 YouTuBi 名称大小写。
- 新增 MIT License。

## 0.2.0 - 2026-06-14

- 新增 YouTube 视频页弹幕开关。
- 支持弹幕随视频播放、暂停和跳转同步。
- 支持鼠标悬停暂停单条弹幕。
- 支持加载并展开评论回复面板。
- 优化回复面板层级展示和字体大小。
- 新增中英文界面支持。
- 新增扩展图标和项目 README。

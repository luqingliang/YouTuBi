# Youtubi

Youtubi 是一个 Chrome 扩展，它会把 YouTube 视频下方的评论变成随视频播放的弹幕。相比反复打开评论区翻看讨论，它希望让观众可以在看视频的同时，自然地看到其他人的即时反应、吐槽和补充。

## 体验重点

- 弹幕会覆盖在 YouTube 播放器上，和视频一起播放。
- 暂停、继续、跳转视频时，弹幕也会跟着同步。
- 视频页面上有一个和 YouTube 风格一致的弹幕开关。
- 可以调整弹幕显示范围、速度、字号、透明度和预加载数量。
- 鼠标悬停在某条弹幕上时，该条弹幕会暂停。
- 如果评论有回复，可以在悬停时展开回复面板，看到上下文关系。
- 支持中文和英文界面。

## 使用方法

目前 Youtubi 暂不发布到 Chrome 插件市场，建议直接下载 zip 包使用。

1. 在 GitHub Releases 页面下载最新的 `youtubi-*.zip`。
2. 将 zip 包解压到本地目录。
3. 打开 Chrome，进入 `chrome://extensions/`。
4. 开启右上角的“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择刚刚解压出来的目录。
7. 打开或刷新 YouTube 视频页。

加载成功后，视频页面点赞按钮左侧会出现弹幕开关。点击浏览器右上角的 Youtubi 图标，可以调整弹幕范围、速度、字号、透明度和预加载数量。

如果你是从源码克隆项目，也可以直接选择项目目录加载。更新插件文件后，需要回到 `chrome://extensions/` 重新加载 Youtubi，然后刷新 YouTube 页面。

## 隐私

Youtubi 只在 YouTube 页面运行。评论数据只用于在当前页面生成弹幕，不会发送到第三方服务器。

## 状态

项目目前仍在快速迭代中，核心目标是先把 YouTube 上的弹幕观看体验打磨顺，再逐步完善更多细节。

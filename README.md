# Youtubi

为 YouTube 带来更好的弹幕体验。

Youtubi 是一个 Chrome 扩展，它会把 YouTube 视频下方的评论变成随视频播放的弹幕。相比反复打开评论区翻看讨论，它希望让观众可以在看视频的同时，自然地看到其他人的即时反应、吐槽和补充。

## 为什么做

YouTube 的评论区内容很多，但它和视频本身是分离的。很多评论其实是围绕视频中的某个瞬间产生的，可是在观看过程中，这些内容不会自然出现。

Youtubi 想补上这个体验：让评论重新回到视频画面里，像弹幕一样跟随播放节奏出现，同时尽量保持可控、干净、不打扰。

## 体验重点

- 弹幕会覆盖在 YouTube 播放器上，和视频一起播放。
- 暂停、继续、跳转视频时，弹幕也会跟着同步。
- 视频页面上有一个和 YouTube 风格一致的弹幕开关。
- 可以调整弹幕显示范围、速度、字号、透明度和预加载数量。
- 鼠标悬停在某条弹幕上时，该条弹幕会暂停。
- 如果评论有回复，可以在悬停时展开回复面板，看到上下文关系。
- 支持中文和英文界面。

## 设计取向

Youtubi 不是简单地把评论随机撒到屏幕上，而是尽量把弹幕做成一种稳定的观看体验：

- 弹幕播放应该像视频一样线性、可暂停、可跳转。
- 默认样式应该融入 YouTube，而不是显得突兀。
- 弹幕内容应该优先来自评论数据本身，页面读取只作为备用方案。
- 回复关系应该能看懂，但不要把面板做成完整评论区。

## 本地体验

1. 打开 Chrome 的 `chrome://extensions/`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目目录。
5. 打开或刷新 YouTube 视频页。

修改代码后，需要在扩展管理页重新加载 Youtubi，并刷新 YouTube 页面。

## 隐私

Youtubi 只在 YouTube 页面运行。评论数据只用于在当前页面生成弹幕，不会发送到第三方服务器。

## 状态

项目目前仍在快速迭代中，核心目标是先把 YouTube 上的弹幕观看体验打磨顺，再逐步完善更多细节。
